Method Route Description
GET / Menampilkan halaman utama berisi menu Customer Product Transaction
GET /customers Menampilkan seluruh Customer
GET /customers/:id Menampilkan 1 Customer
GET /customers/add Menampilkan form add Customer
POST /customers/add Menambahkan data Customer
GET /customers/:id/edit Menampilkan form edit Customer
POST /customers/:id/update Mengubah data Customer
GET /customers/:id/show-cart Menampilkan keranjang Customer
GET /customers/:id/show-transactions Menampilkan seluruh transaksi Customer
POST /customers/:id/payment-confirmation-form Menampilkan form konfirmasi pembayaran
POST /customers/:id/pay-transaction Mengirimkan bukti pembayaran dan mengubah status transaksi menjadi "paid"
GET /products Menampilkan seluruh Product
GET /products/add Menampilkan form add Product
POST /products/add Menambahkan data Product
GET /products/:id/edit Menampilkan form edit Product
POST /products/:id/edit Mengubah data Product
GET /products/:id/delete Menghapus data Product
GET /carts/add Menampilkan form add Cart
POST /carts/add Menambahkan produk kedalam keranjang
GET /carts/:id/delete Menghapus keranjang
GET /transactions Menampilkan seluruh data Transaction
GET /transactions/:id Menampilkan detail dari Transaction
POST /transactions/add Menambahkan data Transaction
GET /transactions/:id/payment-confirmation/accepted Mengubah status transaksi menjadi "accepted"
GET /transactions/:id/payment-confirmation/cancelled Mengubah status transaksi menjadi "cancelled"
GET /transactions/:status Menampilkan seluruh data Transaction berdasarkan status
